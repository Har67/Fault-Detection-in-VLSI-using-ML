# Fault-Detection-in-VLSI-using-ML
Machine Learning-based fault detection project for VLSI circuits using Verilog HDL, Icarus Verilog, GTKWave, Python, and Gradio. The project implements a 4-bit ALU with SA0, SA1, and Random fault analysis, dataset generation, ML-based fault classification, confusion matrix, fault coverage, and browser-based prediction interface.


# Fault Detection in VLSI Circuits using Machine Learning

Machine Learning-based fault detection project for VLSI circuits using Verilog HDL, Icarus Verilog, GTKWave, Python, and Gradio.

## Features
- 4-bit ALU Design using Verilog
- Fault Injection (SA0, SA1, Random Fault)
- Dataset Generation
- ML-based Fault Classification
- Confusion Matrix & Accuracy Analysis
- Gradio-based User Interface

## Tools Used
- Verilog HDL
- Icarus Verilog
- GTKWave
- Python
- Scikit-learn
- Gradio

  ## Main Aim & Future Scope

The main aim of this project is to explore intelligent fault detection techniques in VLSI circuits using Machine Learning concepts integrated with Digital Design and DFT fundamentals.

Currently, the model predicts faults using manually generated input-output datasets from ALU operations and fault simulations.

As a future enhancement, the project can be extended towards automated waveform-based fault identification, where Machine Learning models directly analyze simulation waveforms (VCD/GTKWave data) to detect and classify faults without relying only on manually entered input-output values.

This approach can support advanced DFT and automated semiconductor testing methodologies used in modern VLSI verification and reliability analysis.
