# Fault-Detection-in-VLSI-using-ML
Machine Learning-based fault detection project for VLSI circuits using Verilog HDL, Icarus Verilog, GTKWave, Python, and Gradio. The project implements a 4-bit ALU with SA0, SA1, and Random fault analysis, dataset generation, ML-based fault classification, confusion matrix, fault coverage, and browser-based prediction interface.
